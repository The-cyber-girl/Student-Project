# Student Project

This beginner-friendly Java project demonstrates Object-Oriented Programming (OOP) concepts using a `Student` class.

## Features
- `Student` class with attributes like name, age, grade, and student ID.
- Constructor to initialize student objects.
- Methods to display student information and update grades.
- Demonstrates object creation, method calls, and managing object state.

## How to run
1. Compile `Student.java` and `Main.java`.
2. Run `Main.java` to see example student information and grade updates.

## Example Output
- Shows each student's information.
- Updates grades or other attributes.
- Displays results in a readable format.
